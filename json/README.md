
``` JS Object example
users = {
	"user1": {
    	"firstname": "John",
        "lastname": "Doe"
    }
};
console.log(users['user1']['firstname']);
```

``` PHP Associative array(like object in JS, or dictionary in Python) example
$users = [
	"user1" => [
		"firstname" => "John",
		"lastname" => "Doe",
	]
];

echo $users['user1']['firstname'];
```
